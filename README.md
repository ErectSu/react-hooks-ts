# react-hooks-ts

## 技术栈

1. 脚手架：create-react-app
2. 使用 React Hooks 开发
3. TypeScript
4. Ant Design
5. CSS Modules,
6. Sass

## 集成 Eslint

1. 配置文件：.eslintrc
2. 安装插件 Eslint
3. 安装 prettier
4. 增加 .prettierrc 重置 prettier 配置
5. 执行 `yarn eslint`

## 启动应用

1. 启动代理服务 `yarn start-server`
2. 执行 `yarn start`
3. 浏览器就会自动打开：`http://localhost:3000`
